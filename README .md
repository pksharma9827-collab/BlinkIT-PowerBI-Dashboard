# 📌 Project Overview

This project showcases an interactive **Power BI dashboard** developed
using the **BlinkIT Grocery Sales Dataset**. It transforms raw retail
data into meaningful business insights using KPI cards, DAX, Power
Query, and interactive visualizations.

------------------------------------------------------------------------

# 🎯 Business Objectives

-   Analyze Total Sales
-   Compare Outlet Performance
-   Monitor Customer Ratings
-   Analyze Product Categories
-   Build Executive KPI Dashboard
